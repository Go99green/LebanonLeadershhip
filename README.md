# Lebanon Leadership Project Website

A premium Next.js + Tailwind website for an MBA group project on culture, leadership, and organizational practice in Lebanon.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React

## Local setup
```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## GitHub + Vercel deployment
1. Create a new GitHub repository.
2. Upload all project files.
3. Push to GitHub.
4. In Vercel, click **Add New Project**.
5. Import the GitHub repo.
6. Keep the default Next.js build settings.
7. Deploy.

## Notes
- `Website mode` is for browsing.
- `Presentation mode` is for class delivery with slightly tighter spacing and a more presentation-friendly flow.
- The GLOBE section is presented as a **regional proxy**, not an official Lebanon-only score.
